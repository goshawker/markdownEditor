# MarkdownParser Test Document

## Introduction

This is a test document for the **MarkdownParser** application. It demonstrates various *markdown* features.

## Features

### File Operations

- **Open**: Open any `.md` file via Cmd+O or File > Open
- **Save**: Save your edits with Cmd+S or File > Save
- **Save As**: Save to a new file with Cmd+Shift+S

### Text Formatting

You can use **bold**, *italic*, ~~strikethrough~~, and `inline code`.

## Title Tree Navigation

The title tree in the left sidebar shows all headings.

### Right-Click to Copy

Right-click any heading in the title tree to:
- Copy the heading text
- Copy the heading plus all its sub-content
